The problem

Linux wasn't built
for autonomous AI

Every AI agent today runs on an OS designed for humans. The result: coarse isolation, no native communication, no crash resilience.

001

No fine-grained isolation

A compromised agent in Docker can probe the network, read env vars, access shared volumes. Containers are walls with holes.

002

No native communication

50 agents need Redis, RabbitMQ, or gRPC just to talk. Three external services for basic message passing.

003

No crash resilience

Agent crashes. Nothing restarts it with the right context. Kubernetes restarts pods but doesn't understand supervision trees.

004

Security teams say no

"How do we let AI act autonomously without giving it the keys to the kingdom?" Nobody has a good answer. Until now.

Design principles

Built different

Six principles. No compromise.

01

Zero ambient authority

An agent with no capabilities can do nothing. Every access requires an explicit, unforgeable kernel token.

cap #2 -> Agent(4) [WRITE] // Can message agent 4 ✓ // Cannot spawn ✗ // Cannot network ✗
02

Crash is expected

AI agents fail. The kernel manages supervision trees with automatic restart policies. Like Erlang, for AI.

supervisor (RestartOne) ├── researcher-1 [Running] ├── researcher-2 [Crashed → Restart] └── aggregator [Running]
03

Real hardware

PCI bus discovery. Virtio DMA ring buffers. Split virtqueues. This is a real kernel, not a hosted runtime.

[pci] 00:03.0 vendor=1AF4 virtio-net [net] Queue 0: 256 entries (RX) [net] Queue 1: 256 entries (TX) [net] ONLINE (real driver)
04

Pure Rust. No C.

Memory safety without runtime cost. unsafe only at hardware boundaries. No GC. No runtime. No compromise.

5,384 lines of Rust 0 lines of C 8 unsafe blocks (hw I/O only)
05

Delegation, not escalation

Agents give children subsets of their own capabilities. Never more. Revocation cascades instantly.

supervisor [SPAWN|KILL|DELEGATE] ├── agent-1 [WRITE to agent-3] └── agent-2 [READ from storage] // cannot delegate what // it doesn't have
06

Agent-native kernel

Not a general OS with agents bolted on. Every primitive is designed for autonomous AI workloads.

struct AgentConfig { system_prompt: String, model: ModelBinding, tools: Vec<String>, capabilities: Vec<CapId>, restart_policy: RestartPolicy, }
