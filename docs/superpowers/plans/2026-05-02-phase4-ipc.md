# Phase 4: IPC (Inter-Process Communication) — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement multiple IPC mechanisms — message passing, shared memory grants, request/reply, pub/sub channels, and capability transfer — all gated by the capability system.

**Architecture:** IPC is kernel-mediated. Every send/receive validates capabilities. Messages are kernel-buffered (copied from sender to kernel buffer to receiver). Shared memory uses capability grants for zero-copy. Pub/sub uses named channels with subscriber lists.

**Tech Stack:** Kernel data structures, `alloc` collections, capability validation on every operation.

```rust
// oxide-os/kernel/src/ipc/message.rs
use alloc::collections::VecDeque;
use alloc::collections::BTreeMap;
use alloc::vec::Vec;
use spin::Mutex;
use super::{IpcError, MessageId, MessageType};
use crate::task::TaskId;
use crate::task::scheduler::SCHEDULER;
use crate::capability::{CapId, CAP_TABLE, PermissionBits};
use crate::println;

static NEXT_MSG_ID: Mutex<u64> = Mutex::new(1);

fn next_message_id() -> MessageId {
    let mut id = NEXT_MSG_ID.lock();
    let msg_id = MessageId(*id);
    *id += 1;
    msg_id
}

/// A kernel-buffered message.
#[derive(Debug, Clone)]
pub struct Message {
    pub id: MessageId,
    pub sender: TaskId,
    pub recipient: TaskId,
    pub msg_type: MessageType,
    pub payload: Vec<u8>,
    pub cap_transfer: Option<CapId>,
    pub reply_to: Option<MessageId>,
}

/// Per-task mailbox: a bounded queue of incoming messages.
const MAILBOX_CAPACITY: usize = 256;

pub struct Mailbox {
    queue: VecDeque<Message>,
}

impl Mailbox {
    pub fn new() -> Self {
        Mailbox {
            queue: VecDeque::with_capacity(16),
        }
    }

    pub fn push(&mut self, msg: Message) -> Result<(), IpcError> {
        if self.queue.len() >= MAILBOX_CAPACITY {
            return Err(IpcError::MailboxFull);
        }
        self.queue.push_back(msg);
        Ok(())
    }

    pub fn pop(&mut self) -> Option<Message> {
        self.queue.pop_front()
    }

    pub fn is_empty(&self) -> bool {
        self.queue.is_empty()
    }

    pub fn len(&self) -> usize {
        self.queue.len()
    }
}

/// Global mailbox registry: one mailbox per task.
pub static MAILBOXES: Mutex<BTreeMap<TaskId, Mailbox>> = Mutex::new(BTreeMap::new());

/// Register a mailbox for a new task.
pub fn register_mailbox(task_id: TaskId) {
    MAILBOXES.lock().insert(task_id, Mailbox::new());
}

/// Remove a task's mailbox (on task death).
pub fn unregister_mailbox(task_id: TaskId) {
    MAILBOXES.lock().remove(&task_id);
}

/// Send a message to another task.
/// Requires: sender holds a capability to communicate with recipient.
pub fn send(
    sender: TaskId,
    recipient: TaskId,
    msg_type: MessageType,
    payload: Vec<u8>,
    cap_transfer: Option<CapId>,
    reply_to: Option<MessageId>,
    sender_cap: CapId,
) -> Result<MessageId, IpcError> {
    // Validate capability: sender must have a cap pointing to the recipient agent
    {
        let table = CAP_TABLE.lock();
        table.validate(sender_cap, sender, PermissionBits::WRITE)
            .map_err(|_| IpcError::CapabilityDenied)?;
    }

    let msg_id = next_message_id();
    let msg = Message {
        id: msg_id,
        sender,
        recipient,
        msg_type,
        payload,
        cap_transfer,
        reply_to,
    };

    // Deliver to recipient's mailbox
    let mut mailboxes = MAILBOXES.lock();
    let mailbox = mailboxes.get_mut(&recipient)
        .ok_or(IpcError::RecipientNotFound)?;
    mailbox.push(msg)?;

    // Wake recipient if it was blocked waiting for a message
    let mut sched = SCHEDULER.lock();
    sched.unblock(recipient);

    Ok(msg_id)
}

/// Receive a message (non-blocking).
pub fn receive(task_id: TaskId) -> Option<Message> {
    MAILBOXES.lock().get_mut(&task_id)?.pop()
}

/// Receive a message (blocking — blocks task if no message available).
pub fn receive_blocking(task_id: TaskId) -> Option<Message> {
    // Try to get a message
    let msg = MAILBOXES.lock().get_mut(&task_id).and_then(|mb| mb.pop());
    if msg.is_some() {
        return msg;
    }

    // No message — block the task
    {
        let mut sched = SCHEDULER.lock();
        sched.block_current();
    }

    // When woken up, try again
    MAILBOXES.lock().get_mut(&task_id).and_then(|mb| mb.pop())
}
```

```rust
// oxide-os/kernel/src/ipc/shared_memory.rs
use alloc::collections::BTreeMap;
use alloc::vec::Vec;
use spin::Mutex;
use crate::task::TaskId;
use crate::capability::{CapId, CAP_TABLE, PermissionBits, ResourceRef};
use crate::memory::frame_allocator::FRAME_ALLOCATOR;
use crate::memory::PAGE_SIZE;
use crate::println;
use super::IpcError;

pub type SharedRegionId = u64;

/// A shared memory region that multiple tasks can map.
#[derive(Debug)]
pub struct SharedRegion {
    pub id: SharedRegionId,
    pub phys_base: u64,
    pub size: u64,         // In bytes (page-aligned)
    pub owner: TaskId,
    pub grants: Vec<TaskId>, // Tasks that have been granted access
}

static NEXT_REGION_ID: Mutex<u64> = Mutex::new(1);
pub static SHARED_REGIONS: Mutex<BTreeMap<SharedRegionId, SharedRegion>> = Mutex::new(BTreeMap::new());

/// Create a new shared memory region.
/// Allocates physical frames and returns a region ID.
pub fn create_region(owner: TaskId, size: u64) -> Result<SharedRegionId, IpcError> {
    let page_count = (size + PAGE_SIZE - 1) / PAGE_SIZE;

    // Allocate contiguous frames
    let mut alloc = FRAME_ALLOCATOR.lock();
    let alloc = alloc.as_mut().ok_or(IpcError::RecipientNotFound)?;

    let first_frame = alloc.allocate_frame().ok_or(IpcError::RecipientNotFound)?;
    let phys_base = first_frame.start_address().as_u64();

    // Allocate remaining frames (they may not be contiguous — for simplicity we allocate one by one)
    for _ in 1..page_count {
        alloc.allocate_frame().ok_or(IpcError::RecipientNotFound)?;
    }

    let mut id_lock = NEXT_REGION_ID.lock();
    let id = *id_lock;
    *id_lock += 1;

    let region = SharedRegion {
        id,
        phys_base,
        size: page_count * PAGE_SIZE,
        owner,
        grants: Vec::new(),
    };

    println!("[shm] Created region #{}: {} pages at phys {:#X} (owner: task {})",
        id, page_count, phys_base, owner);

    SHARED_REGIONS.lock().insert(id, region);

    // Create a capability for the owner
    let mut cap_table = CAP_TABLE.lock();
    cap_table.create_root(
        owner,
        ResourceRef::Memory { base: phys_base, length: page_count * PAGE_SIZE },
        PermissionBits::READ.union(PermissionBits::WRITE).union(PermissionBits::DELEGATE),
        None,
    );

    Ok(id)
}

/// Grant access to a shared region to another task.
/// Requires: granter holds a capability with DELEGATE permission for this region.
pub fn grant_access(
    region_id: SharedRegionId,
    granter: TaskId,
    grantee: TaskId,
    granter_cap: CapId,
) -> Result<CapId, IpcError> {
    // Validate granter's capability
    {
        let table = CAP_TABLE.lock();
        table.validate(granter_cap, granter, PermissionBits::DELEGATE)
            .map_err(|_| IpcError::CapabilityDenied)?;
    }

    // Add grantee to region's grant list
    let mut regions = SHARED_REGIONS.lock();
    let region = regions.get_mut(&region_id)
        .ok_or(IpcError::RecipientNotFound)?;
    region.grants.push(grantee);

    // Create a read-only capability for the grantee
    let mut cap_table = CAP_TABLE.lock();
    let cap_id = cap_table.create_root(
        grantee,
        ResourceRef::Memory { base: region.phys_base, length: region.size },
        PermissionBits::READ,
        None,
    );

    println!("[shm] Granted region #{} access to task {} (cap #{})", region_id, grantee, cap_id);
    Ok(cap_id)
}

/// Get physical address of a shared region (for mapping into task's address space).
pub fn get_region_phys(region_id: SharedRegionId) -> Option<(u64, u64)> {
    SHARED_REGIONS.lock().get(&region_id).map(|r| (r.phys_base, r.size))
}
```

```rust
// oxide-os/kernel/src/ipc/channel.rs
use alloc::collections::BTreeMap;
use alloc::string::String;
use alloc::vec::Vec;
use spin::Mutex;
use super::{IpcError, MessageId, message};
use crate::task::TaskId;
use crate::capability::{CapId, CAP_TABLE, PermissionBits};
use crate::println;

pub type ChannelId = u64;

/// A named pub/sub channel.
pub struct Channel {
    pub id: ChannelId,
    pub name: String,
    pub subscribers: Vec<(TaskId, CapId)>, // (subscriber task, their subscribe cap)
}

static NEXT_CHANNEL_ID: Mutex<u64> = Mutex::new(1);
pub static CHANNELS: Mutex<BTreeMap<String, Channel>> = Mutex::new(BTreeMap::new());

/// Create a new named channel.
pub fn create(name: String) -> ChannelId {
    let mut id_lock = NEXT_CHANNEL_ID.lock();
    let id = *id_lock;
    *id_lock += 1;

    let channel = Channel {
        id,
        name: name.clone(),
        subscribers: Vec::new(),
    };

    println!("[channel] Created '{}' (id: {})", name, id);
    CHANNELS.lock().insert(name, channel);
    id
}

/// Subscribe to a channel. Requires SUBSCRIBE capability for the channel.
pub fn subscribe(
    channel_name: &str,
    task_id: TaskId,
    cap_id: CapId,
) -> Result<(), IpcError> {
    // Validate subscribe capability
    {
        let table = CAP_TABLE.lock();
        table.validate(cap_id, task_id, PermissionBits::SUBSCRIBE)
            .map_err(|_| IpcError::CapabilityDenied)?;
    }

    let mut channels = CHANNELS.lock();
    let channel = channels.get_mut(channel_name)
        .ok_or(IpcError::RecipientNotFound)?;

    // Avoid duplicate subscriptions
    if !channel.subscribers.iter().any(|(t, _)| *t == task_id) {
        channel.subscribers.push((task_id, cap_id));
        println!("[channel] Task {} subscribed to '{}'", task_id, channel_name);
    }

    Ok(())
}

/// Unsubscribe from a channel.
pub fn unsubscribe(channel_name: &str, task_id: TaskId) -> Result<(), IpcError> {
    let mut channels = CHANNELS.lock();
    let channel = channels.get_mut(channel_name)
        .ok_or(IpcError::RecipientNotFound)?;

    channel.subscribers.retain(|(t, _)| *t != task_id);
    Ok(())
}

/// Publish a message to all subscribers of a channel.
/// Requires PUBLISH capability for the channel.
pub fn publish(
    channel_name: &str,
    sender: TaskId,
    payload: Vec<u8>,
    publish_cap: CapId,
) -> Result<usize, IpcError> {
    // Validate publish capability
    {
        let table = CAP_TABLE.lock();
        table.validate(publish_cap, sender, PermissionBits::PUBLISH)
            .map_err(|_| IpcError::CapabilityDenied)?;
    }

    let channels = CHANNELS.lock();
    let channel = channels.get(channel_name)
        .ok_or(IpcError::RecipientNotFound)?;

    let subscriber_count = channel.subscribers.len();
    let subscribers: Vec<TaskId> = channel.subscribers.iter().map(|(t, _)| *t).collect();
    drop(channels); // Release lock before sending

    // Deliver message to each subscriber's mailbox
    let mut mailboxes = message::MAILBOXES.lock();
    for &sub_task in &subscribers {
        if sub_task == sender { continue; } // Don't send to self
        if let Some(mailbox) = mailboxes.get_mut(&sub_task) {
            let msg = message::Message {
                id: super::MessageId(0), // Pub/sub messages don't need reply tracking
                sender,
                recipient: sub_task,
                msg_type: super::MessageType::Data,
                payload: payload.clone(),
                cap_transfer: None,
                reply_to: None,
            };
            let _ = mailbox.push(msg);
        }
    }

    // Wake blocked subscribers
    let mut sched = crate::task::scheduler::SCHEDULER.lock();
    for &sub_task in &subscribers {
        sched.unblock(sub_task);
    }

    Ok(subscriber_count)
}
```

```rust
// oxide-os/kernel/src/ipc/request_reply.rs
use alloc::collections::BTreeMap;
use alloc::vec::Vec;
use spin::Mutex;
use super::{IpcError, MessageId, MessageType, message};
use crate::task::TaskId;
use crate::task::scheduler::SCHEDULER;
use crate::capability::{CapId, CAP_TABLE, PermissionBits};
use crate::interrupts;

/// Pending request waiting for a reply.
struct PendingRequest {
    requester: TaskId,
    msg_id: MessageId,
    deadline_tick: u64,
}

static PENDING: Mutex<BTreeMap<u64, PendingRequest>> = Mutex::new(BTreeMap::new());
static REPLIES: Mutex<BTreeMap<u64, message::Message>> = Mutex::new(BTreeMap::new());

/// Send a request and block until reply or timeout.
/// `timeout_ticks` is how many timer ticks to wait before returning Timeout.
pub fn request(
    sender: TaskId,
    recipient: TaskId,
    payload: Vec<u8>,
    sender_cap: CapId,
    timeout_ticks: u64,
) -> Result<message::Message, IpcError> {
    // Validate capability
    {
        let table = CAP_TABLE.lock();
        table.validate(sender_cap, sender, PermissionBits::WRITE)
            .map_err(|_| IpcError::CapabilityDenied)?;
    }

    // Send the request message
    let msg_id = message::send(
        sender,
        recipient,
        MessageType::Request,
        payload,
        None,
        None,
        sender_cap,
    )?;

    let deadline = interrupts::ticks() + timeout_ticks;

    // Register pending request
    PENDING.lock().insert(msg_id.0, PendingRequest {
        requester: sender,
        msg_id,
        deadline_tick: deadline,
    });

    // Block the sender
    {
        let mut sched = SCHEDULER.lock();
        sched.block_current();
    }

    // Check for reply (after being woken)
    if let Some(reply) = REPLIES.lock().remove(&msg_id.0) {
        return Ok(reply);
    }

    // Check timeout
    if interrupts::ticks() >= deadline {
        PENDING.lock().remove(&msg_id.0);
        return Err(IpcError::Timeout);
    }

    Err(IpcError::Timeout)
}

/// Send a reply to a previous request.
pub fn reply(
    sender: TaskId,
    original_msg_id: MessageId,
    payload: Vec<u8>,
    sender_cap: CapId,
) -> Result<(), IpcError> {
    // Find the pending request
    let pending = PENDING.lock().remove(&original_msg_id.0)
        .ok_or(IpcError::RecipientNotFound)?;

    // Store the reply
    let reply_msg = message::Message {
        id: MessageId(0),
        sender,
        recipient: pending.requester,
        msg_type: MessageType::Reply,
        payload,
        cap_transfer: None,
        reply_to: Some(original_msg_id),
    };

    REPLIES.lock().insert(original_msg_id.0, reply_msg);

    // Wake the requester
    let mut sched = SCHEDULER.lock();
    sched.unblock(pending.requester);

    Ok(())
}

/// Called periodically from timer — check for timed-out requests.
pub fn check_timeouts() {
    let current_tick = interrupts::ticks();
    let mut pending = PENDING.lock();

    let expired: Vec<u64> = pending
        .iter()
        .filter(|(_, req)| current_tick >= req.deadline_tick)
        .map(|(id, _)| *id)
        .collect();

    for id in expired {
        if let Some(req) = pending.remove(&id) {
            // Wake the requester so it gets the timeout error
            let mut sched = SCHEDULER.lock();
            sched.unblock(req.requester);
        }
    }
}
```

```rust
/// Send a message with optional capability transfer.
/// If cap_transfer is Some, the capability is moved from sender to recipient.
pub fn send(
    sender: TaskId,
    recipient: TaskId,
    msg_type: MessageType,
    payload: Vec<u8>,
    cap_transfer: Option<CapId>,
    reply_to: Option<MessageId>,
    sender_cap: CapId,
) -> Result<MessageId, IpcError> {
    // Validate sender's communication capability
    {
        let table = CAP_TABLE.lock();
        table.validate(sender_cap, sender, PermissionBits::WRITE)
            .map_err(|_| IpcError::CapabilityDenied)?;
    }

    // If transferring a capability, validate ownership and perform transfer
    if let Some(transfer_cap_id) = cap_transfer {
        let mut table = CAP_TABLE.lock();
        // Validate sender owns the cap being transferred
        let cap = table.get(transfer_cap_id)
            .map_err(|_| IpcError::InvalidCapTransfer)?;
        if cap.owner != sender {
            return Err(IpcError::InvalidCapTransfer);
        }
        // Delegate to recipient (creates a new cap owned by recipient)
        table.delegate(
            transfer_cap_id,
            sender,
            recipient,
            cap.permissions, // Same permissions
            None,            // Same resource
        ).map_err(|_| IpcError::InvalidCapTransfer)?;
    }

    let msg_id = next_message_id();
    let msg = Message {
        id: msg_id,
        sender,
        recipient,
        msg_type,
        payload,
        cap_transfer,
        reply_to,
    };

    let mut mailboxes = MAILBOXES.lock();
    let mailbox = mailboxes.get_mut(&recipient)
        .ok_or(IpcError::RecipientNotFound)?;
    mailbox.push(msg)?;

    let mut sched = SCHEDULER.lock();
    sched.unblock(recipient);

    Ok(msg_id)
}
```
